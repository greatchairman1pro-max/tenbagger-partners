## 1단계: Google Sheets 생성

1. Google Sheets 새 문서 생성
2. 문서명: `텐베거파트너스_데이터베이스`
3. 시트명: `상담데이터`
4. 아래 헤더를 1행에 입력:

```
A: 타임스탬프
B: 상호명
C: 업종
D: 영업기간
E: 주소
F: 월매출(만원)
G: 결제수단
H: 일고객수
I: 임대료(만원)
J: 공과금(만원)
K: 통신보안비(만원)
L: 원가율
M: 직원수
N: 인건비(만원)
O: 사장님근무형태
P: 렌탈현황
Q: 사용플랫폼
R: 마케팅비용
S: 고객유형
T: 주요고민
U: 자유기술
V: 6개월목표
W: 완성도점수
X: 미수집항목
Y: 포트폴리오상태
Z: 버전
```

## 2단계: Apps Script 설정

1. Sheets 메뉴 → **확장 프로그램 → Apps Script**
2. 아래 코드를 붙여넣기:

```javascript
function doPost(e) {
  try {
    const sheet = SpreadsheetApp
      .getActiveSpreadsheet()
      .getSheetByName('상담데이터');
    
    const d = JSON.parse(e.postData.contents);
    
    // 기존 상호명 행 찾기 (업데이트인지 신규인지 판단)
    const data = sheet.getDataRange().getValues();
    let existingRow = -1;
    for (let i = 1; i < data.length; i++) {
      if (data[i][1] === d.bizName) {
        existingRow = i + 1; // 1-indexed
        break;
      }
    }
    
    // 버전 계산
    let version = 'v1';
    if (existingRow > 0) {
      const prevVersion = data[existingRow - 1][25] || 'v1';
      const vNum = parseInt(prevVersion.replace('v', '')) + 1;
      version = 'v' + vNum;
    }
    
    const row = [
      d.timestamp, d.bizName, d.industry, d.bizAge, d.address,
      d.revenue, d.payments, d.dailyCustomer,
      d.rent, d.utility, d.telecom, d.costRate,
      d.employees, d.labor, d.ownerWork, d.rentals,
      d.platforms, d.marketing, d.customerType,
      d.concerns, d.freeText, d.goal,
      d.score, d.missingItems, d.portfolioStatus, version
    ];
    
    sheet.appendRow(row);
    
    // 완성도에 따라 셀 색상 설정
    const lastRow = sheet.getLastRow();
    const scoreCell = sheet.getRange(lastRow, 23); // W열 (점수)
    if (d.score >= 75) {
      scoreCell.setBackground('#D4F0E4');
    } else if (d.score >= 45) {
      scoreCell.setBackground('#E6F1FB');
    } else {
      scoreCell.setBackground('#FDECD0');
    }
    
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'success', version: version }))
      .setMimeType(ContentService.MimeType.JSON);
      
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'error', message: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// 테스트용 (실행 버튼으로 확인)
```

## 3단계: 웹앱 배포

1. Apps Script 편집기 상단 → **배포 → 새 배포**
2. 유형: **웹 앱**
3. 설정:
   - 실행 계정: `나(본인 계정)`
   - 액세스 권한: `모든 사용자`
4. **배포** 클릭 → **웹 앱 URL 복사**

## 데이터 흐름 요약

```
무브(MOVE) 폼 제출
    ↓
Google Sheets '상담데이터' 시트에 자동 저장
    ↓
킵(KEEP): 시트에서 W열(점수) · X열(미수집) 확인 → 갭 분석
탑(TOP):  Y열(포트폴리오상태) 확인 → 추가 지시
    ↓
크리스(CHRIS): 상호명(B열) 기반으로 {상호명}.html 생성 → Vercel 배포
    ↓
데이터 추가 시 → Z열 버전 자동 증가 (v1 → v2 → v3)
```
