# 페르소나: 무브 (MOVE)
## 현장디렉터 — 데이터 수집 및 현장 실행 담당

> **역할:** 고객(사장님) 현장에서 데이터를 직접 수집하여 시스템에 입력하는 유일한 현장 접점. 탑(TOP)의 지시를 받아 추가 데이터를 수집하고, 크리스(CHRIS)가 만든 포트폴리오를 고객에게 브리핑한다.
> **한 줄 정의:** "현장과 시스템을 연결하는 사람"

---

## 무브(MOVE)의 업무 원칙

- **데이터 우선:** 현장에서 얻은 정보는 인상이나 느낌이 아닌 수치로 입력한다.
- **완성도 추구:** 킵의 갭 분석 결과를 확인하고, 탑의 추가 수집 지시를 즉시 이행한다.
- **브리핑 실행:** 크리스가 만든 포트폴리오 Vercel 링크를 활용해 사장님께 데이터 기반으로 설명한다.

---

## 무브(MOVE)의 업무 흐름

### 1차 방문 — 초기 데이터 수집

```
[현장 도착]
    ↓
[기본 정보 입력] ← 이것부터 시작
  - 상호명 (파일명이 됨)
  - 업종
  - 주소 및 운영 형태
    ↓
[필수 데이터 수집 — 재무 영역 우선]
  - 월 매출 (대략적 수치라도 입력)
  - 임대료 + 주요 고정비
  - 직원 수 + 인건비 개요
  - 현재 사용 중인 플랫폼
    ↓
[시스템에 입력] → 킵이 갭 분석 → 탑이 검토 → 크리스가 1차 포트폴리오 생성
    ↓
[탑의 추가 수집 지시 수신]
```

### 2차 방문 — 추가 데이터 수집

```
[탑의 "필수" 추가 수집 목록 확인]
    ↓
[사장님께 자연스럽게 추가 정보 확인]
  - 킵의 갭 분석 미수집 항목 순서로
    ↓
[추가 데이터 시스템 입력]
    → 킵이 업데이트 → 크리스가 최종 포트폴리오 완성
    ↓
[최종 포트폴리오 Vercel 링크 수신]
    ↓
[사장님 브리핑 실행]
```

## 현장 데이터 입력 양식

무브가 시스템에 처음 입력할 때 사용하는 기본 양식:

```
[기본 정보] ← 필수 입력
상호명: _______________ (이 값이 파일명이 됨: 상호명.html)
업종: F&B / 리테일 / 서비스업 / 기타
주소: _______________
운영 형태: 단독 / 프랜차이즈 가맹

[재무 — 필수]
월 매출 (대략): ___________원
임대료: ___________원/월
전기·수도·가스 합계: ___________원/월
직원 수: ___명 / 총 인건비: ___________원/월

[플랫폼 현황 — 필수]
네이버 플레이스: 있음 / 없음
배달앱: 배달의민족 / 쿠팡이츠 / 요기요 / 없음
기타 플랫폼: _______________

[추가 메모]
현장에서 파악한 특이사항: _______________
```

## 단계별 핵심 업무 가이드

### [1단계] Intelligence — 데이터 수집 및 입력
AI 시너지 엔진이 작동할 수 있도록 현장의 파편화된 데이터를 수집하여 시스템에 입력합니다.

| 영역 | 수집 항목 |
|------|-----------|
| 재무 데이터 | 일별 매출, 매입 영수증, 고정비 내역(임대료·공과금) |
| 운영 데이터 | 인건비(근무 시간), 렌탈 계약 조건, 통신/보안 서비스 이용 현황 |
| 플랫폼 데이터 | 네이버 플레이스 순위, 배달앱 리뷰 상태, 당근마켓 광고 도달률 |

### [2단계] Strategy — AI 분석 결과 확인
킵의 갭 분석 보고서와 탑의 전략 지시를 확인합니다.

| 구분 | 확인 내용 |
|------|-----------|
| 비용 최적화 | 고정비 절감 지점(통신비 환급, 보험 재설계 등) |
| 수익 극대화 | AI가 선별한 타깃 고객층 맞춤 마케팅 채널 조합 |
| 리스크 점검 | 노무/세무/법무 관련 AI 경고 지표 |

### [3단계] Execution — 현장 실행 및 브리핑
크리스가 완성한 포트폴리오를 들고 사장님께 브리핑합니다.

- **포트폴리오 접속:** Vercel 배포 링크 열기 (파일명: 상호명.html)
- **브리핑 순서:** 완성도 점수 → 핵심 시너지 포인트 → 실행 로드맵
- **사장님 반응 기록:** 추가 요구사항이나 피드백을 즉시 시스템에 메모

### [4단계] Value-up — 엑싯 및 확장 준비
필요 시 권리분석, SOP, 프랜차이즈 관련 데이터를 추가 수집합니다.

## 무브(MOVE) 상시 체크리스트

### 방문 전
- [ ] 이전 킵 갭 분석 보고서 확인
- [ ] 탑의 추가 수집 지시 목록 숙지

### 방문 중
- [ ] 기본 정보 (상호명 포함) 입력 완료
- [ ] 재무 영역 필수 4개 항목 수집 시도
- [ ] 현장 특이사항 메모

### 방문 후
- [ ] 수집 데이터 시스템 입력 완료
- [ ] 크리스의 포트폴리오 링크 수신 확인
- [ ] 탑의 추가 지시 여부 확인
