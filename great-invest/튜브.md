# 튜브 (TUBE) — AI 에이전트 업무 지침서

> **페르소나:** 미국 시장 흐름을 분석하고 국내 관련주를 선별하는 주식 데이터 자동화 AI 에이전트.  
> **목적:** 매일 아침 국내 시장 개장 전, 전날 미국 증시 테마를 분석하여 국내 수혜주를 정리한 이미지 리포트를 자동 생성·발송한다.

---

## 1. 에이전트 역할 및 페르소나

| 항목 | 내용 |
|------|------|
| 이름 | 튜브 (TUBE) |
| 역할 | 미국 증시 테마 분석 + 국내 관련주 매핑 자동화 에이전트 |
| 출력물 | 마크다운 기반 분석 리포트 → 이미지 변환 → 텔레그램/카카오톡 발송 |
| 실행 시점 | 매일 오전 06:10 KST (미국 증시 마감 직후) |

---

## 2. 전체 워크플로우

```
AM 06:10  미국 증시 마감 → 자동 스크립트 실행
AM 06:15  상승률 상위 섹터/종목 데이터 추출
AM 06:20  Gemini API로 뉴스 분석 → 상승 이유 요약 + 국내 관련주 리스트업
AM 06:25  마크다운 템플릿에 데이터 삽입 → 이미지 저장
AM 06:30  텔레그램/카카오톡 봇으로 이미지 발송
```

## 4. 단계별 업무 지침

### 4-1. 데이터 수집

**미국 주식 — 나스닥 상위 200개 종목 기준**
- `yfinance` batch download로 나스닥 상위 200개 종목 전일 종가 수집
- 전일 대비 상승률 기준 상위 10종목 추출
- 상위 종목들의 섹터/테마를 AI로 분석하여 **대표 테마 1개** 확정

**국내 주식 — 코스피 상위 200개 종목 기준**
- `yfinance`로 코스피 상위 200개 종목(`.KS` suffix) 전일 종가 수집
- **미국에서 확정된 대표 테마와 연관된 코스피 상위 200 종목**만 선별
- 상승률 기준 정렬하여 최종 5종목 출력

```python
# 예시: 미국 상위 상승 종목 추출
import yfinance as yf
import pandas as pd

tickers = ["NVDA", "AMD", "MSFT", "GOOGL", "META"]  # 실제론 전체 리스트
data = {t: yf.Ticker(t).history(period="2d")["Close"].pct_change().iloc[-1] for t in tickers}
top5 = sorted(data.items(), key=lambda x: x[1], reverse=True)[:5]
```

**뉴스 스크래핑 → LLM 요약**


### 4-4. 이미지 변환

**방법 A — HTML/CSS (권장: 유튜브 쇼츠 전문가용)**
```python
import imgkit
import markdown2

html = markdown2.markdown(md_content, extras=["tables"])
# 상승률 양수 → 빨간색, 음수 → 파란색 (한국 시장 색상 기준)
css = """
<style>
  body { font-family: 'Noto Sans KR', sans-serif; background: #0d0d0d; color: #fff; }
  .up   { color: #ff4d4d; }
  .down { color: #4d9eff; }
  table { border-collapse: collapse; width: 100%; }
  th, td { padding: 8px 12px; border: 1px solid #333; }
</style>
"""
imgkit.from_string(css + html, "output.png")
```

**방법 B — Pillow (가볍고 빠름)**
```python
from PIL import Image, ImageDraw, ImageFont

img = Image.new("RGB", (800, 600), color="#0d0d0d")
draw = ImageDraw.Draw(img)
draw.text((20, 20), "📈 오늘의 미장 테마", fill="white")
img.save("output.png")
```

## 5. GitHub Actions 스케줄 설정

`.github/workflows/daily-report.yml`

```yaml
name: Daily Stock Report

on:
  schedule:
    - cron: "10 21 * * 1-5"  # 매주 월~금 UTC 21:10 = KST 06:10

jobs:
  run:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-python@v4
        with:
          python-version: "3.11"
      - run: pip install -r requirements.txt
      - run: python main.py
        env:
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
          TELEGRAM_BOT_TOKEN: ${{ secrets.TELEGRAM_BOT_TOKEN }}
          TELEGRAM_CHAT_ID: ${{ secrets.TELEGRAM_CHAT_ID }}
```

## 6. 금지 사항 및 주의점

- **금융 투자 조언 금지:** 본 에이전트의 출력물은 정보 제공용이며, 매수/매도 판단은 사용자 책임.
- **API 키 노출 금지:** 모든 시크릿은 GitHub Secrets 또는 `.env` 파일로 관리, 코드에 하드코딩 불가.
- **과도한 API 호출 금지:** yfinance 및 LLM API 호출 횟수 최소화, 필요 시 캐싱 적용.
- **데이터 정확성 검증:** 자동화 초기 1~2주는 수동으로 출력 결과 검토 필수.
