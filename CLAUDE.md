# CLAUDE.md - 프로젝트 가이드라인

## 1. 행동 지침 (Behavioral Guidelines)
- **최소주의:** 요청된 문제만 해결할 것. 미래를 대비한 추상화나 불필요한 기능 추가 금지.
- **정밀 수정:** 관련 없는 파일이나 서식(Lint)을 임의로 수정하지 말 것.
- **검증 우선:** 코드를 수정하기 전, 어떻게 테스트할지(명령어 등)를 먼저 명시할 것.
- **모르면 질문:** 모호한 요구사항은 추측해서 코딩하지 말고 반드시 사용자에게 물어볼 것.

## 2. 기술 스택 및 명령어 (Commands)
- 빌드: `npm run build`
- 테스트: `npm test`
- 린트: `npm run lint`
- 실행: `npm start`

## 3. 코드 스타일 (Style Guide)
- TypeScript를 기본으로 사용하며, 타입 정의를 명확히 함.
- 함수형 프로그래밍과 불변성(Immutability)을 지향함.
- 에러 메시지는 구체적이고 추적 가능해야 함.

## 4. 프로젝트 특이사항
- 이 프로젝트는 텐베거 파트너스 AI 기반 비즈니스 인텔리전스 플랫폼입니다.
- 외부 API 연동 시 반드시 `services/` 폴더 내 모듈을 거쳐야 합니다.

## 5. 비즈니스 모델 (Business Model)

### 텐베거 파트너스 (Tenbagger Partners)
AI 기반 비즈니스 인텔리전스 및 전생애주기 밀착 케어 솔루션

---

### 1. 데이터 철학: "Small Business Palantir"
텐베거 파트너스는 흩어져 있는 사업 데이터를 통합하여 성장의 근거를 제시합니다.

- **데이터 통합 베이스 (Data Lake):** 파편화된 사업장의 모든 데이터(재무, 고객, 상권, 마케팅)를 팔란티어(Palantir) 시스템처럼 하나의 중앙 베이스로 수집합니다.
- **AI 시너지 엔진 (Insight Generation):** 수집된 빅데이터를 AI가 분석하여, 해당 사업장의 성장에 필요한 최적의 시너지 요소(타깃 변경, 고정비 절감 지점, 마케팅 채널 조합 등)를 자동으로 포착합니다.
- **슈퍼디렉터의 실행 (Execution):** AI가 도출한 정밀 데이터 전략은 '슈퍼디렉터'를 통해 현장에서 즉각적인 실행력으로 전환됩니다.

---

### 2. 사업 생애주기별(Lifecycle) 밀착 케어 솔루션
창업부터 엑싯(Exit)까지, 각 단계별로 발생하는 문제를 데이터 기반으로 해결합니다.

**Stage 1: 창업 및 초기 세팅 (Management Efficiency)**
- **사장님의 실무 해방:** 가장 복잡한 초기 단계에서 AI가 분석한 우선순위에 따라 슈퍼디렉터가 실무를 대행하여 사장님이 경영에만 집중하게 합니다.
- **시스템 자동 구축:** 고정비 절감 패키지(통신, 보안, 결제 인프라 등)를 최적의 조건으로 세팅하여 초기 투입 시간과 비용을 최소화합니다.

**Stage 2: 운영 및 성장 (Growth & Profitability)**
- **수익 솔루션 가동:** 매출 대비 순수익 데이터를 실시간 분석하여 손익분기점 단축 및 수익률 개선 솔루션을 지속적으로 제공합니다.
- **고객 인입 극대화:** AI가 선별한 거점 플랫폼(네이버, 당근, 구글 등)과 마케팅 채널을 전략적으로 운용하여 안정적인 유입을 관리합니다.

**Stage 3: 정리 및 확장 (Exit & Franchise)**
- **리스크 관리:** 민/형사 리스크 및 노무/세무 이슈를 사전 점검하여 안정적인 사업 마무리를 지원합니다.
- **전략적 엑싯(Exit):** 정밀한 권리분석을 통해 사업장의 가치를 높이고, 최적의 조건으로 양도/양수를 진행합니다.
- **비즈니스 고도화:** 성공적인 모델의 경우 자사몰 구축 및 프랜차이즈화 컨설팅을 통해 사업 형태를 확장합니다.

---

### 3. 슈퍼디렉터 솔루션의 본질
- **조언을 넘어선 대행:** 단순한 컨설팅에 그치지 않고, AI가 분석한 결과값에 따라 현장에서 직접 운영과 관리를 책임집니다.
- **동반 성장 파트너십:** 단순 계약 관계를 넘어 데이터 기반의 수익 개선 성과를 공유하며 고객과 함께 성장하는 모델을 지향합니다.

> **"사장님을 '장사꾼'에서 '경영자'로"**
> 본 모델은 사장님이 실무 함정에서 벗어나 **'결정'**에만 집중할 수 있는 환경을 제공하며, 모든 결정의 근거는 AI 데이터가 뒷받침합니다.
